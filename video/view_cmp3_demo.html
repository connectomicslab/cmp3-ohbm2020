Connectome Mapper 3 - Software Demonstration

Author: Sebastien Tourbier

Date: 2020 June 10

This video shows how to use the graphical user interface of the Connectome Mapper 3 to easily (1) create the pipeline configuration files, (2) configure and execute of the BIDS App run, and (3) visually inspect the stage outputs.
